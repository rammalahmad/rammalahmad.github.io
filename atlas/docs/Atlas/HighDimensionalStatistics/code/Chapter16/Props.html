    Chapter 16: Spectral Theory of Bounded Operators #

    This file formalizes Propositions 16.1, 16.2, and 16.3 from Chapter 16 on the spectral theory of bounded linear operators on Hilbert spaces.

    Main results #

    • prop_16_1 — Proposition 16.1 (Goal 110): The spectrum of a bounded linear operator on a Hilbert space is a compact subset of {|z| ≤ ‖T‖}.
    • prop_16_2 — Proposition 16.2 (Goal 111): For a bounded self-adjoint operator A with m = inf_{‖φ‖=1}⟨Aφ, φ⟩ and M = sup_{‖φ‖=1}⟨Aφ, φ⟩, {m, M} ⊆ spec(A) ⊆ [m, M].
    • prop_16_3 — Proposition 16.3 (Goal 112): For a bounded self-adjoint operator A and a real polynomial p, ‖p(A)‖ ≤ sup_{t ∈ [m,M]} |p(t)|.

    References #

    These results are standard in the spectral theory of bounded operators on Hilbert spaces; see e.g. Reed and Simon, Methods of Modern Mathematical Physics I: Functional Analysis, Chapter VII.
