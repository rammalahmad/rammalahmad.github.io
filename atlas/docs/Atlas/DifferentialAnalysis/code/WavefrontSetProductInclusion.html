    def WavefrontSet.WFscMinkowskiSum {n : ℕ} (u v : TemperedDistribution (EuclideanSpace ℝ (Fin n)) ℂ) :
    Set (ClosedBall n × ClosedBall n)

    Minkowski-style upper bound for the scattering wavefront set of a product/sum of distributions: a pair (p, q) lies in this set if (p, q) ∈ WFsc u, or (p, q) ∈ WFsc v, or there exist boundary points p₁, p₂ with (p₁, pq.2) ∈ WFsc u and (p₂, pq.2) ∈ WFsc v while (p, q) lies on the boundary product.
