    def MeasuresAndSigmaAlgebras.IsPositiveLinearFunctional {X : Type u_1} [MetricSpace X] (u : ZeroAtInftyContinuousMap X ℝ →L[ℝ] ℝ) :
    Prop

    A continuous linear functional u : C₀(X, ℝ) →L[ℝ] ℝ is positive if u f ≥ 0 whenever f ≥ 0 pointwise.
