    theorem WavefrontSet.not_mem_wfsc_iff_exists_css_decomp {n : ℕ} {u : TemperedDistribution (EuclideanSpace ℝ (Fin n)) ℂ} {p q : ClosedBall n} (hpq : (p, q) ∈ BoundaryProd n) :
    (p, q) ∉ WFsc u ↔ ∃ (u₁ : TemperedDistribution (EuclideanSpace ℝ (Fin n)) ℂ) (u₂ : TemperedDistribution (EuclideanSpace ℝ (Fin n)) ℂ), u = u₁ + u₂ ∧ p ∉ Css u₁ ∧ q ∉ Css (FourierTransform.fourier u₂)

    Equivalence (Melrose Thm 12.18-style): (p, q) ∉ WFsc u iff u decomposes into pieces avoiding p in physical and q in Fourier conic singular support.
