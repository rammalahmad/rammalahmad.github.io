    theorem DifferentialOperators.reciprocal_poly_deriv_bound_multiIndex {n : ℕ} (P : MvPolynomial (Fin n) ℂ) (m : ℕ) (hm : 1 ≤ m) (hdeg : P.totalDegree ≤ m) (C : ℝ) (hP : HasPolyLowerBound P m C) (js : List (Fin n)) :
    ∃ Cα > 0, ∀ (ξ : EuclideanSpace ℝ (Fin n)), ‖ξ‖ > 1 / C → ‖iteratedPartialDeriv js (polyReciprocal P) ξ‖ ≤ Cα * ‖ξ‖ ^ (-↑(m + js.length))

    Multi-index derivative bound for 1/P outside {‖ξ‖ > 1/C}: under the polynomial lower bound ‖P(ξ)‖ ≥ C · ‖ξ‖^m, an iterated partial derivative ∂^α (1/P) of order |α| decays like ‖ξ‖^(-(m + |α|)). This is the elliptic-style parametrix decay estimate from Melrose Cor 12.15.
