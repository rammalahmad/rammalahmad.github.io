    def IsSmooth (B n : ℕ) :
    Prop

    A natural number n is B-smooth if all of its prime factors are at most B, i.e. n ∈ Nat.smoothNumbers (B + 1). Smoothness is the key notion underlying subexponential factoring algorithms.
