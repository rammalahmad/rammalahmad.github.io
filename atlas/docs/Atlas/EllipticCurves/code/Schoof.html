    ↑c = ↑t

    Lemma 8.2: let E/𝔽_q have Frobenius π, let ℓ be a prime with ℓ ∤ q, and let P ∈ E[ℓ] be nonzero. If for some integer c the equation π_ℓ²(P) - c · π_ℓ(P) + q_ℓ · P = 0 holds, then c ≡ tr π (mod ℓ). This is the core congruence underlying Schoof's algorithm: any integer c solving the characteristic equation on a single nonzero ℓ-torsion point must agree with the trace of Frobenius modulo ℓ.
