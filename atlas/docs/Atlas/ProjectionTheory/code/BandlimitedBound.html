          Bandlimited bound (Lemma, "Fourier method in Euclidean space"). If supp ĝ ⊆ B_{1/r}, then |g(x)|² ≲ (|g|² ∗ ψ_r)(x) for some non-negative approximate identity ψ_r at scale r.
