                                  Orthogonality of tube wave packets (Lemma): if two $1 \times R$ tubes $T_1, T_2$ do not fit into a common $R^\varepsilon r \times R^{1 + \varepsilon}$ rectangle, then $|\langle \phi_{T_1, r}, \phi_{T_2, r}\rangle| \lesssim R^{-1000}$.
