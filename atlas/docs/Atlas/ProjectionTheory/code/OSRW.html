        Orponen–Shmerkin–Ren–Wang theorem. For X ⊂ ℝ² and s < dimH X, the set of directions whose projection of X has Hausdorff dimension < s is itself small: $$\dim_H\{\theta : \dim_H(\pi_\theta X) < s\} \le 2s - \dim_H X.$$
