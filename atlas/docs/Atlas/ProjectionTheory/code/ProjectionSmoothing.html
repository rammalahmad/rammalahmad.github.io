    ℝ → ℂ

    The projection π_θ f : ℝ → ℂ of an L² function f : ℝ^d → ℂ along the direction θ ∈ S^{d−1}: at the point t ∈ ℝ it returns the integral of f over the affine hyperplane t·θ + θ^⊥, with respect to the Haar measure on θ^⊥.
