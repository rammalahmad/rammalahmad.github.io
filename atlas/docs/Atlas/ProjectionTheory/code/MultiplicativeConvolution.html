    noncomputable def mulConv (f g : ℕ → ℂ) (n : ℕ) :
    ℂ

    The multiplicative (Dirichlet) convolution of two functions $f, g : \mathbb{N} \to \mathbb{C}$: $(f *_M g)(n) = \sum_{n_1 n_2 = n} f(n_1)\, g(n_2)$.
