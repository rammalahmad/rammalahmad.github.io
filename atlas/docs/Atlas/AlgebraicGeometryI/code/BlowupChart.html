    noncomputable def blowupChartMap (k : Type u_1) [CommRing k] :
    Polynomial (Polynomial k) →+* Polynomial (Polynomial k)

    Ring map on k[X][Y] substituting Y ↦ C(X) · Y, modelling one affine chart of the blow-up of the affine plane at the origin (where the exceptional y-coordinate is replaced by the product of the base coordinate X and a new fibre coordinate Y).
