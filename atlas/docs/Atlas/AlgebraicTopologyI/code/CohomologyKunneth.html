    abbrev CohomologyKunneth.totalSingularCohomology (R : Type) [CommRing R] (X : TopCat) :
    Type

    Total singular cohomology of X with coefficients in R, assembled as the direct sum ⨁_{n : ℕ} H^n(X; R) over natural-number degrees.
