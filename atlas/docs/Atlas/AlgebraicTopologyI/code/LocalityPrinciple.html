                                  theorem LocalityPrinciple.locality_principle (X : TopCat) (𝒜 : Set (Set ↑X)) (h𝒜 : IsCover 𝒜) [∀ (i : ℕ), HomologicalComplex.HasHomology (smallSingularChainComplex X 𝒜) i] [∀ (i : ℕ), HomologicalComplex.HasHomology (AlgebraicTopologyI.singularChainComplexZ X) i] :
                                  QuasiIso (smallSingularInclusion X 𝒜)

                                  The Locality Principle (Theorem 11.4 / 13.5). For any cover $\mathcal{A}$ of a space $X$, the inclusion of $\mathcal{A}$-small chains $S_*^{\mathcal{A}}(X) \subseteq S_*(X)$ is a quasi-isomorphism; equivalently, the induced map $H_*^{\mathcal{A}}(X) \to H_*(X)$ is an isomorphism in every degree.
