    structure ComplexDifferentialFormAlgebra (Ω : ℕ → Type u_1) (VF : Type u_2) [inst : DifferentialFormSpace Ω VF] (R : Type u_3) [CommRing R] [Algebra ℂ R] [IsGradedFormAlgebra R] :
    Type (max u_1 u_3)

    A $\mathbb{C}$-algebra $R$ realizing the graded algebra of complex differential forms: each $\Omega^p$ embeds $\mathbb{R}$-linearly into a homogeneous component of degree $p$ in $R$, with an inverse extraction map.

    • embed (p : ℕ) : Ω p → R
    • embed_linear (p : ℕ) (a b : Ω p) (c : ℝ) : self.embed p (c • a + b) = (algebraMap ℝ R) c * self.embed p a + self.embed p b
    • embed_homog (p : ℕ) (α : Ω p) : IsGradedFormAlgebra.IsHomog (self.embed p α) p
    • extract (p : ℕ) : R → Ω p
    • extract_embed (p : ℕ) (α : Ω p) : self.extract p (self.embed p α) = α
