    Positivity of the dihedral iterate along an alternating word: for any $n < m(s, t)$ (or $m(s, t) = \infty$), the action of the alternating word $\mathtt{alternatingWord}\,s\,t\,n$ on $e_s$ via $\sigma$ produces a positive root.
