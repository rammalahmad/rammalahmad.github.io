    GL (Fin n) C.k

    The transvection / elementary matrix $E_{ij}(c) = 1 + c \cdot e_{ij}$ packaged as an element of $\mathrm{GL}_n(k)$, with explicit two-sided inverse $E_{ij}(-c)$.
