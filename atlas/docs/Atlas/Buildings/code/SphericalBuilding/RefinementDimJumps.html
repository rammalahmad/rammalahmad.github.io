    def GLnBuilding.CompleteFlag.IsStrictFlag {k : Type u_1} [Field k] {n : ℕ} (σ : CompleteFlag k n) :

    A strict flag: a complete flag $\sigma$ in which each subspace $V_j$ has dimension exactly $j$, i.e., $\dim V_j = j$ for all $j$.
