    def CoxeterComplex.ChamberAdjacent {B : Type u_1} (M : CoxeterMatrix B) (w w' : M.Group) :
    Prop

    Two elements $w, w'$ of the Coxeter group $W$ are chamber-adjacent when $w \neq w'$ and $w' = w \cdot s_i$ for some simple generator $s_i$.
