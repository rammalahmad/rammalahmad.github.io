    ∃ (rf : K.ReversibleFolding), Finset.image rf.f.morph.toFun C = C ∧ Finset.image rf.f.morph.toFun D = C ∧ Finset.image rf.g.morph.toFun D = D ∧ Finset.image rf.g.morph.toFun C = D

    Strong form of the Coxeter reversible folding: for any adjacent pair $(C, D)$, there is a reversible folding $(f, g)$ with $f(C) = C$, $f(D) = C$, $g(D) = D$, $g(C) = D$.
