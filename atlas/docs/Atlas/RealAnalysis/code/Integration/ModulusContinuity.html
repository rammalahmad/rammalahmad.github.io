    noncomputable def Integration.modulusOfContinuity (f : ℝ → ℝ) (a b η : ℝ) :
    ℝ

    The modulus of continuity of a function f on [a, b] at scale η: w_f(η) = sup {|f(x) - f(y)| : x, y ∈ [a,b], |x - y| ≤ η}.

      structure Integration.TaggedPartition.IsTaggedRefinement {a b : ℝ} (T T' : TaggedPartition a b) :
      Type

      T' is a tagged refinement of T when every subinterval of T' is contained in some subinterval of T (witnessed by assign), and the subintervals of T' assigned to each T-subinterval telescope to its length. This captures the condition x ⊂ x' in Theorem II.

      • assign : Fin T'.n → Fin T.n
      • contained_left (j : Fin T'.n) : T.points (self.assign j).castSucc ≤ T'.points j.castSucc
      • contained_right (j : Fin T'.n) : T'.points j.succ ≤ T.points (self.assign j).succ
      • telescope (k : Fin T.n) : ∑ j : Fin T'.n with self.assign j = k, (T'.points j.succ - T'.points j.castSucc) = T.points k.succ - T.points k.castSucc
