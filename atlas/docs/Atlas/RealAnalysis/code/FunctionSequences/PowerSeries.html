    structure FunctionSequences.PowerSeries :
    Type

    A real power series about a center point x₀, given by a sequence of real coefficients a : ℕ → ℝ. Formally, it represents the formal series ∑ₘ aₘ (x - x₀)^m.

    • a : ℕ → ℝ
    • x₀ : ℝ
