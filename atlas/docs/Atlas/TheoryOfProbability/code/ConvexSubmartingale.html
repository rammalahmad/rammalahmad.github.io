    Convex function of a martingale is a submartingale (Lecture 26/27 claim).

    If Xₙ is a martingale w.r.t. the filtration ℱ, and φ : ℝ → ℝ is convex with E|φ(Xₙ)| < ∞ for all n, then φ ∘ Xₙ is a submartingale. The proof uses conditional Jensen's inequality together with the martingale property E[Xⱼ | ℱᵢ] = Xᵢ.
