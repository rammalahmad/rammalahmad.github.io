    theorem wald_equation {Ω : Type u_1} {m : MeasurableSpace Ω} {μ : MeasureTheory.Measure Ω} [MeasureTheory.IsProbabilityMeasure μ] {X : ℕ → Ω → ℝ} (hX_iid : ∀ (i j : ℕ), ProbabilityTheory.IdentDistrib (X i) (X j) μ μ) (hX_ind : ProbabilityTheory.iIndepFun X μ) (hX_int : MeasureTheory.Integrable (X 0) μ) {f : MeasureTheory.Filtration ℕ m} {T : Ω → ℕ} (hT : MeasureTheory.IsStoppingTime f fun (ω : Ω) => ↑(T ω)) (hXf : ∀ (k : ℕ), ProbabilityTheory.Indep (MeasurableSpace.comap (X k) inferInstance) (↑f k) μ) (hT_int : MeasureTheory.Integrable (fun (ω : Ω) => ↑(T ω)) μ) :
    ∫ (ω : Ω), ∑ i ∈ Finset.range (T ω), X i ω ∂μ = (∫ (ω : Ω), X 0 ω ∂μ) * ∫ (ω : Ω), ↑(T ω) ∂μ

    Wald's equation. Let X_i be i.i.d. integrable real random variables and T a stopping time with E T < ∞, with each X_k independent of the filtration at time k. Then E[S_T] = E[X_1] · E[T], where S_T = ∑_{i < T} X_i.
