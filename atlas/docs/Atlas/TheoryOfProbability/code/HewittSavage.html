    def IsExchangeableEvent {S : Type u_2} [MeasurableSpace S] (A : Set (ℕ → S)) :
    Prop

    A set A ⊆ (ℕ → S) is an exchangeable (or permutable) event if it is measurable and invariant under every finite permutation of the index set ℕ, i.e. {x : x ∘ σ ∈ A} = A for every permutation σ that moves only finitely many indices.

      def IsExchangeableEventWrt {Ω : Type u_1} {S : Type u_2} [MeasurableSpace S] (X : ℕ → Ω → S) (E : Set Ω) :
      Prop

      An event E ⊆ Ω is exchangeable with respect to the sequence X : ℕ → Ω → S if there is some exchangeable event A ⊆ (ℕ → S) such that E is the pullback of A under ω ↦ (n ↦ X n ω). Equivalently, E belongs to the σ-field of exchangeable events generated by X.
