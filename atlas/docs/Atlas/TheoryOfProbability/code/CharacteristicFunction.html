    noncomputable def ProbabilityTheory.charFun (μ : MeasureTheory.Measure ℝ) (t : ℝ) :
    ℂ

    The characteristic function of a probability measure μ on ℝ, defined by φ(t) = ∫ exp(i t x) dμ(x). For a random variable X with law μ this is φ_X(t) = E[exp(i t X)].
