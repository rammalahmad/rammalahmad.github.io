    def ProbabilityTheory.IsLatticeRV (μ : MeasureTheory.Measure ℝ) :
    Prop

    A probability measure μ on ℝ corresponds to a lattice random variable when its characteristic function charFun μ equals 1 at some nonzero l (so μ is supported on a translate of (2π/l)ℤ) but is not identically 1 (so μ is not a point mass at 0).
