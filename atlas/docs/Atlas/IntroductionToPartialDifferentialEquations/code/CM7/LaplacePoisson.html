    structure LaplacePoisson.IsLipschitzDomain {n : ℕ} (Ω : Set (Fin n → ℝ)) :
    Prop

    A bounded Lipschitz domain in $\mathbb{R}^n$: an open, bounded, nonempty, connected set whose boundary can be locally straightened by bi-Lipschitz maps. This is the natural class of domains for the Poisson Dirichlet problem.

    • isOpen : IsOpen Ω
    • isBounded : Bornology.IsBounded Ω
    • nonempty : Ω.Nonempty
    • isConnected : IsConnected Ω
    • frontier_nonempty : (frontier Ω).Nonempty
    • lipschitz_boundary (x : Fin n → ℝ) : x ∈ frontier Ω → ∃ (U : Set (Fin n → ℝ)), IsOpen U ∧ x ∈ U ∧ ∃ (φ : (Fin n → ℝ) → Fin n → ℝ), LipschitzWith 1 φ ∧ ∃ (ψ : (Fin n → ℝ) → Fin n → ℝ), LipschitzWith 1 ψ ∧ (∀ y ∈ U, ψ (φ y) = y) ∧ ∀ z ∈ φ '' U, φ (ψ z) = z
