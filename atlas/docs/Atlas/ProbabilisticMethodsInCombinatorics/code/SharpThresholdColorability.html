          theorem SharpThresholdColorability.achlioptas_friedgut (k : ℕ) (hk : k ≥ 3) :
          ∃ (d_k : ℕ → ℝ), (∀ ε > 0, ∀ (d : ℕ → ℝ), (∀ (n : ℕ), d n > 0) → (∀ᶠ (n : ℕ) in Filter.atTop, d n < d_k n - ε) → Filter.Tendsto (fun (n : ℕ) => probColorable n k (d n / ↑n)) Filter.atTop (nhds 1)) ∧ ∀ ε > 0, ∀ (d : ℕ → ℝ), (∀ (n : ℕ), d n > 0) → (∀ᶠ (n : ℕ) in Filter.atTop, d n > d_k n + ε) → Filter.Tendsto (fun (n : ℕ) => probColorable n k (d n / ↑n)) Filter.atTop (nhds 0)

          Theorem 4.3.17 (Achlioptas-Friedgut 2000). For every $k \ge 3$ there is a sharp threshold sequence $d_k(n)$ for $k$-colorability in $G(n, d/n)$: if $d_n < d_k(n) - \varepsilon$ eventually then $\mathbb{P}(G(n, d_n/n) \text{ is } k\text{-colorable}) \to 1$, while if $d_n > d_k(n) + \varepsilon$ eventually then the probability $\to 0$.
