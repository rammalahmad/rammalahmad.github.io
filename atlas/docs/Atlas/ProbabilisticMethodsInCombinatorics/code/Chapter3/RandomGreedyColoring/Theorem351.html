    theorem TwoColorableHypergraph.radhakrishnan_srinivasan_two_colorable :
    ∃ c > 0, ∀ (V : Type u_2) [Fintype V] [DecidableEq V] (H : Hypergraph V) (k : ℕ), 2 ≤ k → IsKUniform H k → ↑(Finset.card H) ≤ c * √(↑k / Real.log ↑k) * 2 ^ k → IsTwoColorable H

    Theorem 3.5.1 (Radhakrishnan–Srinivasan 2000). There is an absolute constant $c > 0$ such that every $k$-uniform hypergraph with at most $c \sqrt{k/\log k} \, 2^k$ edges is 2-colorable.
