          V → Bool

          The greedy 2-coloring induced by an injective ordering $\sigma$: process vertices in increasing $\sigma$-order and color $v$ false if there is an edge $e \ni v$ whose other vertices are smaller than $v$ and already colored true; otherwise color $v$ true.

            Structural lemma for random greedy coloring. If an injective vertex ordering $\sigma$ on a $k$-uniform hypergraph $H$ (with $k \geq 2$) has no conflicting pair, then the greedy coloring induced by $\sigma$ witnesses that $H$ is 2-colorable.
