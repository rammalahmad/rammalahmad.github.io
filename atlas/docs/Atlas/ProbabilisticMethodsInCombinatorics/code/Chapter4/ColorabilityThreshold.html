      theorem ColorabilityThreshold.sharp_threshold_k_colorability (k : ℕ) :
      k ≥ 3 → ∃ d_k > 0, (∀ d > 0, d < d_k → Filter.Tendsto (fun (n : ℕ) => probKColorable k n d) Filter.atTop (nhds 1)) ∧ ∀ d > d_k, Filter.Tendsto (fun (n : ℕ) => probKColorable k n d) Filter.atTop (nhds 0)

      $k$-colorability sharp threshold (Achlioptas–Friedgut, Theorem 4.3.17): for every $k \ge 3$ there exists $d_k > 0$ such that for $d < d_k$ the graph $G(n, d/n)$ is $k$-colorable whp, while for $d > d_k$ it is not.
