    def HypergraphColoring.goodColorings624 {V : Type u_1} (edges : Set (Finset V)) (X : Finset V) :
    Set (V → Bool)

    Set of 2-colorings of V that properly color every hyperedge e ∈ edges contained in the finite vertex set X (no edge entirely monochromatic). Used in the compactness step of Theorem 6.2.4.

      Theorem 6.2.4: a (possibly non-uniform, possibly infinite) hypergraph in which every edge has size at least $3$ and satisfies the LLL weight bound $\sum_{f} 2^{-|f|} \le 1/8$ over its intersecting neighbors admits a proper 2-coloring.
