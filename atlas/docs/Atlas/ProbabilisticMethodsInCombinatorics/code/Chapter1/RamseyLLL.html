    (Theorem 1.1.9, Spencer 1977 — Ramsey lower bound via LLL) If $(\binom{k}{2}\binom{n}{k-2} + 1) \cdot 2^{1 - \binom{k}{2}} < 1/e$, then there exists a graph on $n$ vertices with no $k$-clique in $G$ or $G^c$, witnessing $R(k, k) > n$.
