            theorem TwoColorableHypergraph.erdos_non_two_colorable_upper_bound :
            ∃ (C : ℝ), 0 < C ∧ ∀ (k : ℕ), 2 ≤ k → ∃ (E : Hypergraph (Fin (k ^ 2))), IsKUniform E k ∧ ↑(Finset.card E) ≤ C * ↑k ^ 2 * 2 ^ k ∧ ¬IsTwoColorable E

            (Theorem 1.3.3) $m(k) = O(k^2 \, 2^k)$: there is a constant $C > 0$ such that for every $k \ge 2$ there is a $k$-uniform non-2-colorable hypergraph with at most $C k^2 2^k$ edges.
