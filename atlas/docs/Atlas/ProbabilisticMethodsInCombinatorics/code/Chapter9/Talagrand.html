          def Talagrand.IsMedian {n : ℕ} (μ : MeasureTheory.Measure (Fin n → ℝ)) (f : (Fin n → ℝ) → ℝ) (m : ℝ) :
          Prop

          $m$ is a median of $f$ under $\mu$ if both $\mu(\{f \leq m\}) \geq 1/2$ and $\mu(\{f \geq m\}) \geq 1/2$.
