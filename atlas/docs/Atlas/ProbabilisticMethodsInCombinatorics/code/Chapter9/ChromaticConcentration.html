    def ChromaticConcentration.DiffOnlyAt {V : Type u_1} (G G' : SimpleGraph V) (v : V) :
    Prop

    Two graphs $G$ and $G'$ on vertex set $V$ differ only at vertex $v$: their adjacency relations agree on all pairs not involving $v$.
