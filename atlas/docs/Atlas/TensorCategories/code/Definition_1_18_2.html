    Definition 1.18.2: A k-linear abelian category C is finite if (i) C has finite dimensional spaces of morphisms; (ii) every object of C has finite length; (iii) C has enough projectives, i.e., every simple object of C has a projective cover; and (iv) there are finitely many isomorphism classes of simple objects.
