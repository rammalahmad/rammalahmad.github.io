    An algebra in a multitensor category C is a triple (A, m, u) where A is an object of C and m : A ⊗ A → A, u : 1 → A are morphisms (multiplication and unit) satisfying associativity and unit axioms. Implemented as the Mathlib type Mon C of monoid objects.
