    Theorem 1.15.8: In a ring category with right duals, if the unit object 𝟙_ C is indecomposable then it is simple. This is the part (i) of the theorem from Etingof–Gelaki–Nikshych–Ostrik, stating semisimplicity of the unit in the indecomposable case.
