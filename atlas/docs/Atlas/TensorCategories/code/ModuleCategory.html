    Definition 2.1.1 (EGNO): A left module category over a monoidal category C is a category M equipped with a bifunctorial action ⊗ : C × M → M, an associator natural isomorphism (X ⊗ Y) ⊗ N ≅ X ⊗ (Y ⊗ N), and a unit functor 𝟙_C ⊗ - that is an equivalence, satisfying the pentagon coherence axiom.
