    Proposition 1.48.2: A finite tensor category C is integral (i.e. its Frobenius–Perron dimension function is integer-valued) if and only if it admits a quasi-fiber functor; this is equivalent to C being the representation category of a finite-dimensional quasi-Hopf algebra.
