    structure CategoryTheory.FPdimCategoryData (C : Type u) [Category.{v, u} C] [MonoidalCategory C] :
    Type (max u (u_1 + 1))

    Data needed to compute the Frobenius-Perron dimension of a multitensor category C: a finite nonempty index set I, a family of simple objects simpleObj : I → C, an FP-dimension function on C, and the assumption that each simple has FP-dimension at least one.

    • I : Type u_1
    • instFintype : Fintype self.I
    • instNonempty : Nonempty self.I
    • simpleObj : self.I → C
    • fpDimFun : FPdimFunction
    • fpDim_simpleObj_ge_one (i : self.I) : self.fpDimFun.fpDim (self.simpleObj i) ≥ 1
