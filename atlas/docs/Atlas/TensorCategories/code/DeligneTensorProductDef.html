              Definition 1.46.1: Deligne's tensor product C ⊠ D. A witness consisting of a k-linear abelian category tensorCat together with a right-exact bilinear bifunctor ⊠ : C × D ⥤ tensorCat that is universal among such bifunctors out of C × D. Any right-exact bilinear bifunctor F : C × D ⥤ A factors through ⊠ via an essentially unique right-exact k-linear functor F_bar : tensorCat ⥤ A.
