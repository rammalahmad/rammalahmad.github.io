    structure SweedlerH4 (k : Type u_1) [Field k] :
    Type u_1

    Concrete model of Sweedler's 4-dimensional Hopf algebra H₄ over a field k: an element is recorded as a tuple of 4 coefficients in the basis (1, g, x, gx).

    • coeff : Fin 4 → k
